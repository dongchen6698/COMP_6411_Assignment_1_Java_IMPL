Befor use Eclipse to run Kruskal.java. Please set the args[] from Run Configurations --> Arguments --> Program arguments.Like below:
10 AdjacencyMatrix_of_Graph_G_10.txt

args[0] = the number of Vertex	(Like 10)
args[1] = Adjacency Matrix file	(Like AdjacencyMatrix_of_Graph_G_10.txt)

when calculate the data of 10000, try to input the VM arguments like below:

VM Arguments: -Xmx4000m
